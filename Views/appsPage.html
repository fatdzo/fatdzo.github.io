Anchorman FREE

Responsibilities: Application development and design, testing and support, deployment and online promotion.

“Anchorman” is a news headline aggregator and article webpage parser. The article contents are being read using the Android’s built-in text-to-speech engine. Additionally, you have an option to input your custom article url that you would like to listen.

The latest version can be downloaded at the google play store: play.google.com/store/apps/details?id=markomarks.core.anchorman

Skills: Java, Android, JSoup

TabsOnTally

Responsibilities: Application development and design, Team lead

“TabsOnTally” is an open source android project that started at the "Tabs on Tallahassee" hackathon organized by the “Orlando Sentinel” and “Code For Orlando” group. The task is to build a digital tool using the Tabs on Tallahassee API that makes data from the Florida House and/or Senate easier to access and that will help everyday citizens find legislation that matters to them.

The project source code: github.com/fatdzo/TabsOnTally
More information about the API can be found at tabsontallahassee.com/api/

Skills: Java, Android, Ion

JumboGames

Responsibilities: Android app and Web client development, testing, software design

This project was declared the winner of “Hack The Arena” event organized by the “Orlando Tech Association”, “Orlando Magic” and the “Orlando Solar Bears”. “JumboGames” allows sports teams to invite their fans to compete for prizes by playing trivia games on the Jumbotron using their mobile devices as a game controller.

MelodyGenerator

Responsibilities: Application development and design, testing and support, deployment

“MelodyGenerator” produces melodies with backing chords. It uses MusicXML and MIDI as an output format so that the generated melodies can be used as an input to the user’s DAW recording program of choice. The main idea is to help hobby and professional musicians in their music creation process.

Skills: C#, midi-dot-net, Winforms, MusicXML
